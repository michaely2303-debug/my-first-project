# 데이터 분석 프로젝트 포트폴리오

서울시 공공데이터를 활용한 재무/신용 리스크 분석 프로젝트 2건입니다.
두 프로젝트 모두 "어떤 재무 특성이 리스크와 관련 있는가"라는 문제의식을 공유합니다.

## 1. 20대 1인가구 금융 고위험군 분류
`가구_집계구별_생활금융_통계_수정본.ipynb`

- **목표**: 20대 1인가구의 소득 대비 소비율을 기준으로 고위험군을 정의하고, 이를 예측하는 분류 모델 구축
- **접근**: 단순 회귀 대신, 소득 대비 소비율(소비/소득)이 높을수록 상환 여력이 낮아질 것이라는 가설을 세우고 파생변수 설계 → 상위 25%를 고위험군으로 라벨링
- **모델**: RandomForest, LightGBM 비교
- **주의**: 라벨 생성에 사용한 컬럼(평균월소비, 평균총소득)은 피처에서 제외해 데이터 누수를 방지했습니다.

## 2. 집계구별 신용대출 보유 여부 예측
`서울시데이터분석_수정본.ipynb`

- **목표**: 재무 특성(소득 구성, 카드 이용 패턴 등)으로 신용대출 보유자 비중이 높은 집계구를 예측
- **과정**: 처음에는 성별을 타겟으로 시도했으나 정확도가 랜덤 수준(약 55~58%)에 그쳐, 데이터가 실제로 담고 있는 신용/대출 정보를 활용하는 방향으로 타겟을 재정의
- **모델**: DecisionTree, RandomForest, XGBoost 비교 + GridSearch 하이퍼파라미터 튜닝

## 실행 방법

```bash
pip install pandas numpy scikit-learn lightgbm matplotlib seaborn
```

각 노트북 상단의 `pd.read_csv('data/...')` 경로에 맞게 `data/` 폴더에 원본 CSV를 넣고 실행하세요.
